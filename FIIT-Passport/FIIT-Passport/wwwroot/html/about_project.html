Примеры

Название проекта

Сервис для создания комиксов с помощью Kandinsky и GigaChat

Описание проекта

Роли:
- 1 тимлид/PM
- 1 аналитик (либо совмещение с ролью тимлида/PMа)
- 1-2 бэкендера
- 1-2 фронтендера/дизайнер
Всего 3-5 человека
Со стороны заказчика есть куратор, который будет формировать итоговый образ продукта и направлять команду к нему.

Название проекта

Skipper

Описание проекта

Технологии:
- для фронтенда - React JS / Angular
- для бэкенда - GO, Java/Kotlin (Spring framework)
Место развертывания бэкэнда не имеет особого значения (можно использовать как бесплатные, так и лицензионные решения, если такие имеются).

Название проекта

Logica: Задачник по Логическому Программированию

Описание проекта

Роли:

(реальный человек может совмещать/делить несколько ролей)
